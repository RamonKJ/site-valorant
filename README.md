# Valorant Fan Site 🔴

Site estático sobre o jogo **Valorant**, desenvolvido como projeto da disciplina **Introdução à Tecnologia Web** — UNIRIOS, 2026/1.

## 📁 Estrutura de Pastas

```
valorant_site/
│
├── images/          → imagens do site
├── css/
│   └── style.css    → estilos globais
├── js/
│   └── main.js      → jQuery 
├── pages/
│   ├── sobre.html   → Sobre o jogo
│   ├── agentes.html → Agentes e classes
│   ├── mapas.html   → Mapas do jogo
│   ├── armas.html   → Arsenal de armas
│   └── contato.html → Formulário de contato
└── index.html       → Página principal
```

## 📄 Páginas

| Página | Descrição |
|--------|-----------|
| `index.html` | Home com hero, stats e carrossel de agentes |
| `sobre.html` | História e linha do tempo do Valorant |
| `agentes.html` | Cards de agentes com filtro por classe |
| `mapas.html` | Cards com os principais mapas |
| `armas.html` | Tabela completa de armas |
| `contato.html` | Formulário de contato funcional |

## 🛠 Tecnologias

- **HTML5** — estrutura semântica
- **CSS3** — variáveis, flexbox, grid, responsividade
- **jQuery 3.7.1** — interações e animações
